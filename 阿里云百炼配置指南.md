# 阿里云百炼 - 通义千问3-Max 配置指南

## 一、开通服务

1. 访问阿里云百炼平台：https://bailian.console.aliyun.com/
2. 登录/注册阿里云账号
3. 开通"通义千问"服务
4. 创建应用并获取 API Key

## 三、配置步骤

### 1. 获取API Key

1. 进入百炼控制台
2. 点击"应用管理"
3. 创建新应用
4. 复制生成的 API Key

### 2. 修改 application.yml

```yaml
ai:
  enabled: true
  provider: aliyun
  aliyun:
    api-key: sk-xxxxxxxxxxxxxxxxxxxxxxxx  # 替换为您的API Key
    model: qwen-plus  # 或 qwen-max
```

### 3. 模型选择建议

**推荐使用：qwen-plus**
- 性价比最高
- 适合文本分类任务
- 响应速度快

**高端选择：qwen-max**
- 性能最强
- 适合复杂理解任务
- 价格稍高但仍很便宜

## 四、使用示例

### API调用示例

```bash
curl -X POST https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "model": "qwen-plus",
  "input": {
    "messages": [
      {
        "role": "user",
        "content": "请分析这个教学资料并推荐分类：Python编程入门教程.pdf"
      }
    ]
  }
}'
```

### 响应示例

```json
{
  "output": {
    "text": "Python编程",
    "finish_reason": "stop"
  },
  "usage": {
    "total_tokens": 150
  }
}
```

## 五、成本控制

### 月度预算建议

**小型项目（<5000次分类）**
- 预算：5-10元/月
- 可支持：约5000-10000次分类

**中型项目（5000-20000次分类）**
- 预算：10-20元/月
- 可支持：约10000-20000次分类

**大型项目（>20000次分类）**
- 预算：20-50元/月
- 可支持：约20000-50000次分类

### 节省成本技巧

1. **使用缓存**
   - 对相同内容的分类结果缓存
   - 减少重复调用

2. **批量处理**
   - 一次请求处理多个文件
   - 降低API调用次数

3. **智能降级**
   - 先用规则引擎快速匹配
   - 无法识别时再调用AI

## 八、最佳实践

### 1. 混合策略

```java
// 先用规则引擎，再用AI
if (ruleEngine.canClassify(fileName)) {
    return ruleEngine.classify(fileName);

    return aiService.classify(fileName, content);
}
```

### 2. 缓存策略

```java
// 使用Redis缓存分类结果
@Cacheable(value = "classification", key = "#fileName + #contentHash")

    // AI分类逻辑
}
```

### 3. 异步处理

```java
// 大文件异步处理
@Async
public CompletableFuture<Long> classifyAsync(String fileName, String content) {
    return CompletableFuture.completedFuture(
        aiService.classify(fileName, content)
    );
}
```

## 九、故障排查

### 问题1：API调用失败

**可能原因：**
- API Key错误
- 网络问题
- 额度不足

**解决方案：**
```bash
# 检查API Key
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation

# 查看余额
# 登录百炼控制台查看
```

### 问题2：响应慢

**优化方案：**
- 使用qwen-plus代替qwen-max
- 启用缓存
- 减少输入内容长度

### 问题3：成本超预算

**控制方案：**
- 启用缓存机制
- 使用规则引擎优先
- 设置每日调用上限
